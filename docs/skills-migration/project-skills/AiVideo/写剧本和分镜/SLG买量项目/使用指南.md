# SLG 买量广告项目目录

## 快速开始

输入以下内容，即可启动 SLG 买量广告脚本生成：

```
/skill seedance-storyboard-generator

我需要制作一个 [题材] SLG 手游的买量广告视频，
游戏核心卖点是 [卖点]，请帮我生成广告脚本。
```

技能会自动进入**模式 B（SLG买量广告模式）**，依次引导你确认参数。

---

## 六大创意类型速查

| 类型 | 适用场景 | 核心逻辑 |
|------|---------|---------|
| 副玩法钩子型 | 拓展泛用户 | 轻休闲→策略转变 |
| 末日生存剧情型 | 末日/丧尸题材 | 危机感 + 情感代入 |
| 史诗战争型 | 任何战争题材 | 纯视觉震撼 |
| 英雄展示型 | 武将收集向 | 英雄魅力 + 技能冲击 |
| 真人/AI剧情型 | 高代入感需求 | 现实场景 → 游戏解决问题 |
| 挑战失败诱导型 | 益智/策略感强 | 心理挑战欲激发 |

## 题材选择

- **三国** — 最受中国市场欢迎，武将辨识度高
- **末日丧尸** — 全球爆款题材，情感共鸣强
- **史诗战争（中世纪）** — 出海主力题材（占35%）
- **魔幻奇幻** — 视觉自由度高，女性玩家接受度好

## 文件命名规范

```
[游戏代号]_广告脚本_[题材]_[类型简称].md
[游戏代号]_素材清单_广告版.md
[游戏代号]_分镜_[时长].md
```

示例：`征战天下_广告脚本_三国_英雄展示型.md`
