# 3组可套用提示词框架

> 适用于 Seedance 2.0 / 可灵AI，可直接作为分镜脚本骨架，按实际项目替换 [ ] 内的内容。

---

## 框架一：甜蜜爱情

**适用场景**：恋爱短剧、CP互动、节日告白、浪漫宣传片

**情绪弧线**：相遇 → 试探 → 心动 → 确认

```
【风格】温暖胶片质感，高光柔焦，暖黄/粉紫色调，轻微颗粒感
【时长】15秒
【画幅】9:16 竖屏（短视频平台优先）

0-3秒：[两人初遇场景]，阳光/暖光从侧后方打来，人物轮廓发光；
       镜头从[A的视角]缓慢平移至[B]，背景虚化，浅景深，空气感

3-7秒：[小互动触发]——[对视/轻碰/递出物品]，双人中景；
       镜头轻微推近，捕捉[A的表情变化]——从平静到微笑，瞳孔特写

7-11秒：[心动瞬间强化]，慢动作（0.5x速度），[发丝飘动/衣角摆动]；
        暖色光晕扩散，背景虚化加深，画面周围轻微过曝

11-15秒：[关系确认动作]——[牵手/低头一笑/靠近]，双人近景；
         镜头缓缓拉远，阳光逆光，两人剪影，淡出或定格

【声音】轻钢琴 + 吉他指弹，节奏舒缓；关键心动瞬间加心跳音效（若隐若现）
【对白示例】（可选）"[自然生活化的一句话，不超过10字]"
【参考】@图片1 [角色A形象]，@图片2 [角色B形象]，@图片3 [场景参考]
```

**HOOK变体（前3秒替换方案）**：
- 版本A：两人手机屏幕上的聊天记录特写，最后一条消息是"[悬念文字]"，手指点击发送
- 版本B：[A]偷偷看向[B]，[B]恰好转头，对视0.5秒，[A]立刻低头，嘴角压不住
- 版本C：第三人称旁白入画："我以为我不会再喜欢任何人了，直到[那天]..."

## 框架二：复仇爽剧

**适用场景**：逆袭短剧、大女主/大男主叙事、职场/宫斗/江湖复仇

**情绪弧线**：至暗（受辱）→ 蛰伏（积蓄力量）→ 爆发（反击）→ 终局（翻身）

```
【风格】高对比度，暗部压深，青冷底色 + 局部暖光聚焦主角；
       受辱段偏灰偏冷，复仇段转暖金色，象征权力易位
【时长】15秒（可拆分为每集15s，共3-5集）
【画幅】9:16 竖屏

0-3秒：【受辱瞬间】[反派]的压迫动作特写——[扇脸/踩踏/当众羞辱]；
       低角度仰视[反派]，俯视[主角]跌落；[主角]表情：不是哭，是压着怒

3-7秒：【转折符号】[主角]捡起/抓住某个[象征物品]——[残破物件/证据/信物]；
       镜头聚焦在[主角]眼睛上，特写，眼神从绝望变为冰冷；
       BGM从压抑转为低鸣，预示反击

7-11秒：【实力揭示/反击启动】[主角]完成蜕变/获得力量后出场；
        与第一段形成对称——此时[主角]站高处，[反派]在低处；
        服装/气质颠覆，慢动作走来，风吹衣角，逆光强化

11-15秒：【终局一击】[关键对话或动作]，一句台词或一个动作完成复仇；
         [反派]崩溃/失势的表情特写；[主角]转身离开，不回头；
         定格 + 字幕（可选）："[该你了]"

【声音】前段：低沉压抑BGM + 嘲笑/羞辱环境音；
       后段：鼓点爆发型音效 + 大气弦乐起势
【台词节奏】受辱段：沉默或一句狠话（≤8字）；反击段：零台词靠动作
【参考】@图片1 [主角初始状态]，@图片2 [主角蜕变后状态]，@图片3 [核心场景]
```

**HOOK变体（前3秒替换方案）**：
- 版本A：倒叙开场——先展示[主角巅峰时刻]，字幕"三年后"，切回[最惨时刻]
- 版本B：旁白独白："他们以为踩进尘埃的我，会就此消失。"
- 版本C：[反派]得意的笑脸特写，突然画面停格，[主角]OS："记住这张脸。"

## 框架三：悬疑推理

**适用场景**：悬疑短剧、推理解谜、惊悚反转、烧脑剧情

**情绪弧线**：异常 → 调查 → 假象（观众被误导）→ 真相反转

```
【风格】低饱和，冷蓝/墨绿底色，强阴影，手持轻抖；
       线索出现时短暂焦点拉伸（模拟注意力聚焦）
【时长】15秒（作为单集钩子，结尾留悬念）
【画幅】16:9（横屏，营造压迫感）

0-4秒：【异常植入】看似普通的[日常场景]中埋入一个[视觉异常]；
       [人物]没有注意到，但镜头停留在[异常处]0.5秒；
       观众先于主角发现线索，制造"你快看！"的紧迫感

4-8秒：【调查展开】[人物]发现异常，开始观察；
       快速切换：[线索A特写] → [人物眼睛] → [线索B特写] → [人物手部动作]；
       节奏加快，BGM进入，信息密度拉满

8-12秒：【假象构建（误导观众）】[人物]得出错误结论 / 观众以为真相已揭晓；
        镜头给出"确认感"——[人物]放松神态 或 "罪犯"露出笑容；
        BGM短暂松弛，让观众降低防备

12-15秒：【反转一击】一个细节否定了刚才的结论；
         快速特写[颠覆性证据]，[人物]表情骤变；
         BGM骤停 or 弦乐刺耳一声；
         黑场 + 字幕（可选）："[那么，真正的[X]是谁？]"

【声音】全程：低频嗡鸣底噪；线索出现：短促弦乐刺点；反转：乐器骤停+沉默0.3s
【禁忌】不要前4秒就展示最强悬念（留到结尾）；线索不超过2个（太多=乱，太少=无聊）
【参考】@图片1 [主要人物形象]，@图片2 [核心场景]，@图片3 [关键道具/线索物]
```

**HOOK变体（前3秒替换方案）**：
- 版本A：极速闪回——0.5秒内快速剪辑[3个矛盾线索]，最后定格"[你有没有发现？]"
- 版本B：主角直接对镜头说："我知道你是谁了。"然后转身走向[某人]
- 版本C：监控视角俯拍，画面底部时间码快速跳动，锁定某个[可疑动作]

## 使用说明

1. **直接套用**：将 [ ] 替换为实际项目内容，其余文字可保留作为 Seedance 提示词
2. **搭配素材清单**：每个框架对应的 @图片 引用需在素材清单（C/S/P）中提前编号
3. **集数衔接**：每集结尾的"悬念收尾"配合 Seedance 视频延长功能，用尾帧驱动下一集
4. **质量评分**：套用后的分镜脚本仍需通过 6.5分 质量卡口，海外用户审美标准
