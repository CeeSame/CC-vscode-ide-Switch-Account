| 项目 | 限制 | 备注 |

## 二、提示词公式

### 文生视频（主公式）

```
（运镜 + 光影） + 主体（外观描述） + 主体动作 + 场景（场景描述） + 氛围关键词
```

**示例：**
```
镜头缓缓推近，金色侧光，
身穿绿色战袍红色披风的古代将领站立于战场，
缓缓转头望向远方，神情坚毅，
背景为大漠黄沙与破碎的战旗，
三国史诗电影风格，金铜色调
```

### 图生视频（简化公式）

> 图生视频提示词只需描述**运动**，画面由参考图决定

```
主体动作 + 场景变化（如有）+ 氛围/节奏词
```

**示例：**
```
将领缓缓转身面向镜头，眼神从低垂到抬起，
背景战场旗帜随风飘动，
慢镜头，史诗感
```

## 三、运镜关键词

### 文生视频可用（提示词描述 OR 界面选择）

| 运镜效果 | 提示词写法 | 界面选项名 |
|---------|----------|-----------|
| 推近 | 镜头缓缓推近 / 推镜头 | 推进 |
| 拉远 | 镜头缓缓拉远 | 拉远 |

| 垂直移动 | 镜头缓缓上升 / 镜头缓缓下降 | 垂直运镜 |

| 环绕旋转 | 镜头绕主体环绕一圈 | 旋转摇镜 |

## 四、质量保底关键词

加在提示词末尾，防止画面崩坏变形：

```
人物结构正常，面部清晰不变形，五官稳定，
动作流畅不僵硬，画面稳定不抖动，
4K高清，细节清晰
```

## 五、动作描述注意事项

| 要点 | 正确做法 | 错误做法 |
|------|---------|---------|
| 速度 | 缓慢、轻柔、平稳 | 高速、急速、剧烈扭动 |
| 复杂度 | 单一动作，5-10s内完成 | 一段提示词描述多个连续动作 |
| 数量 | 大批骑兵、数名战士 | 50个骑兵、100名士兵 |
| 形容词 | 具体描述（"面容刚毅"） | 主观形容（"很帅"、"超酷"） |
| 物理效果 | 旗帜随风飘动、火焰轻微摇曳 | 球弹跳三次后落地 |

## 六、SLG 广告场景专项禁忌

| 场景 | 禁用表达 | 替代方案 |
|------|---------|---------|
| 战争场面 | 大量流血、肢体分离 | 战斗冲突、交锋、击败 |
| 兵器 | 枪械型现代武器细节描述 | 冷兵器、战略武器、先进装备 |
| 人数 | 具体大数字（100、1000） | 大批、无数、千军万马（氛围词） |
| 表情 | "痛苦扭曲" | 坚毅、激昂、沉着 |
| 历史人物 | 真实政治人物姓名 | 虚构武将名称 |

## 七、15秒广告分段方案

可灵单次最长 10s，15秒广告需分段生成后在剪辑软件合并。

### 推荐方案 A：5s + 10s
```
Clip 1（5秒）文生视频：纯 HOOK，氛围建立
Clip 2（10秒）图生视频：主体展示 + 高潮 + CTA
首帧 = Clip 1 截图末帧 → 确保衔接流畅
```

### 推荐方案 B：10s + 5s
```
Clip 1（10秒）图生视频：HOOK + 主体展示 + 技能
Clip 2（5秒）文生视频：战场高潮 + CTA落版
```

### 方案 C：5s + 5s + 5s（精细控制）
```
Clip 1（5秒）：HOOK
Clip 2（5秒）：主体展示
Clip 3（5秒）：高潮 + CTA
每段首帧 = 上一段截图末帧
```

## 八、视频续写衔接技巧

1. 生成 Clip 1 完成后，截取**最后一帧**保存为图片
2. 用该截图作为 Clip 2 的**首帧参考图**（图生视频模式）
3. 提示词开头写："承接上一场景，[继续描述动作]"
4. 保持风格关键词一致（如"三国史诗电影风格"）
