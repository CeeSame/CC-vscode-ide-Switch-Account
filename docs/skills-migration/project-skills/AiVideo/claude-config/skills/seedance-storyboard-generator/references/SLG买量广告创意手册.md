## 二、六大核心创意类型

### 类型 1：副玩法钩子型（Mini-game Hook）
**原理：** 用轻休闲玩法（跑酷/消消乐/合并/拼图）抓住泛用户，中段过渡到 SLG 核心玩法
**适用题材：** 全题材通用，末日/战争最常见
**脚本结构：**
```
0-3s  [HOOK]  展示休闲副玩法 — "拦住敌人！" / 跑酷突破防线 / 消除障碍
3-9s  [发展]  玩法难度升级 → 引入策略元素（建造/训练/部署）
9-12s [高潮]  宏大战争场面 / 城池被攻破 / 英雄登场
12-15s [CTA]  "立即下载" + 游戏 Logo + 下载按钮
```

**Seedance 提示词关键词：** 动感节奏、快速切换、视觉反差

---

### 类型 2：末日生存剧情型（Post-Apocalyptic Drama）
**原理：** 沉浸式末日叙事制造紧张感，玩家代入"唯一守护者"角色
**代表游戏：** State of Survival、Last Shelter（守望黎明）、Age of Z
**脚本结构：**
```
0-3s  [HOOK]  城市废墟/丧尸浪潮/末日黑云 — 极致危机感开场（无 Logo 无旁白）
3-6s  [冲突]  幸存者困境 / 基地被围攻 / 资源告急
6-9s  [转机]  英雄出现 / 军队集结 / 战略部署
9-12s [高潮]  反击场面：炮火、防线、英雄技能爆发
12-15s [胜利+CTA]  基地崛起 / "率领人类最后的希望，立即加入！"
```

**视觉关键词：** 末日废墟、血色天空、浓烟、幸存者营地、丧尸群、装甲车、爆炸

---

### 类型 3：史诗战争型（Epic War Cinematic）
**原理：** 电影级战争场面震撼视觉，突出策略宏大感
**代表游戏：** Rise of Kingdoms、Evony、万国觉醒
**脚本结构：**
```
0-3s  [HOOK]  千军万马对阵全景 / 攻城器械推进 / 将军策马 — 无对白纯视觉冲击
3-6s  [战场]  步兵方阵推进 / 骑兵冲锋 / 城墙上守军
6-9s  [英雄]  己方英雄登场 — 特写脸部、拔剑/张弓/施法
9-12s [决战]  双方交锋高潮 — 快切镜头、特效爆发
12-15s [CTA]  "你的帝国，等待建立！" + 游戏 Logo
```

**视觉关键词：** 古代战场、旗帜飞扬、盔甲反光、攻城投石车、箭雨、城墙崩塌

---

### 类型 4：英雄展示型（Hero Showcase）
**原理：** 突出明星英雄的视觉魅力和技能，刺激收集欲
**适用题材：** 三国、魔幻、战争
**脚本结构：**
```
0-3s  [HOOK]  黑屏/浓雾 → 英雄登场特写（慢镜头出场）
3-6s  [身份]  角色名字浮现 / 历史/故事背景字幕 / 经典台词 VO
6-9s  [技能]  技能特效展示 1-2 个（视觉冲击最强）
9-12s [战绩]  横扫战场 / 斩杀 BOSS / 部队跟随
12-15s [CTA]  "召唤 [英雄名]，征战天下！" + 立即下载
```

**视觉关键词：** 丁达尔光效、英雄轮廓、技能粒子特效、慢动作

---

### 类型 5：真人/AI剧情型（Live-Action Drama）
**原理：** 真实感最强，代入感最高，现已广泛用 AI 视频替代真人拍摄
**常见套路：**
- 男主营救女主（末日/战争背景）
- 弱→强成长路线
- 反转剧情（貌似失败→绝地反击）

**脚本结构：**
```
0-3s  [HOOK]  危机场景：女主被困/基地被攻/男主无路可退
3-6s  [困境]  情绪渲染：眼神特写、绝望表情、紧张音效
6-9s  [转机]  发现游戏/战略手段 → "只要我们联合起来"
9-12s [反转]  反击成功，形势逆转
12-15s [CTA]  拥抱/胜利 + "现在就加入我们！"
```

**AI 制作注意：** 人物表情自然度关键，推荐真实感风格（Photorealistic）

---

### 类型 6：挑战/失败诱导型（Challenge/Fail Hook）
**原理：** 利用心理反差：展示"显而易见应该成功"却失败，激发挑战欲
**脚本结构：**
```
0-3s  [HOOK]  字幕："99% 的玩家无法守住这座城市" / 展示惨败场面
3-6s  [挑战]  简单策略失败 → 观众心理："我来我能行"
6-9s  [高手]  转入正确策略：英雄配置/资源调配/联盟援助
9-12s [逆转]  城市守住，敌军溃败
12-15s [CTA]  "你能做到吗？立即挑战！"
```

## 三、HOOK 钩子库（前 3 秒开场设计）

> 原则：无 Logo、无旁白、纯视觉张力。每种 HOOK 类型适配不同情绪

### 视觉冲击型
- 万军压境：敌方军队从地平线涌来，尘烟滚滚
- 城市沦陷：现代城市/古代城池在烈焰中崩塌
- 宇宙/太空：舰队从跃迁点涌出，星球爆炸
- 巨龙降临：翼展遮天，阴影笼罩大地

### 情感代入型
- 孤胆英雄：一人面对千军，背影镜头
- 最后防线：战士握紧武器，身后是城市/家园
- 将死之人：英雄跌倒，镜头推近眼神
- 战友情义：两人互相扶持，誓死并肩

### 悬疑好奇型
- 未知力量：黑暗中的眼睛/手/轮廓，缓慢浮现
- 神秘预言：古老地图/卷轴/密码，镜头拉近
- 禁忌之地：迷雾笼罩的废墟，摄像机探入

### 挑战激励型
- 字幕挑战：大字："你能统治这片大陆吗？"
- 倒计时：基地即将被攻破，进度条红色告急
- 玩家评语：截图 "这游戏真的绝了！"

## 四、题材专项视觉规范

### 三国题材
```
统一风格前缀（Nana Banana Pro）：
"Cinematic Three Kingdoms style, ancient Chinese military aesthetics,
ink wash painting mixed with realistic rendering, bronze and crimson color palette"

必备视觉元素：
- 战旗：诸侯旗帜（魏/蜀/吴/其他势力）
- 武器：长枪、青龙偃月刀、弓箭
- 盔甲：汉代军事铠甲，金属质感
- 场景：长坂坡/赤壁/官渡/虎牢关
- 英雄：关羽（赤色/长髯）、吕布（紫金冠）、诸葛亮（羽扇纶巾）
```

### 末日丧尸题材
```
统一风格前缀：
"Post-apocalyptic cinematic style, desaturated color grading with orange glow,
realistic destruction, military survival aesthetics"

必备视觉元素：
- 废墟：倒塌建筑、锈迹车辆、破碎玻璃
- 氛围：血色日落、浓烟、雾气
- 人物：幸存者装备（战术背心、防毒面具）
- 丧尸群：从远景推进，人数感
- 基地：木墙/混凝土防线、岗楼、灯光
```

### 史诗战争题材（欧洲中世纪）
```
统一风格前缀：
"Epic medieval warfare style, cinematic lighting, realistic metal armor,
dramatic sky with storm clouds, 8K ultra-detailed"

必备视觉元素：
- 城堡：哥特式石堡、护城河、瞭望塔
- 军队：骑士骑兵、步兵方阵、攻城器械
- 英雄：铠甲特写、剑盾、斗篷
- 战场：草原/雪地/森林
```

### 三国之外的古风（其他中国历史题材）
```
统一风格前缀：
"Ancient Chinese imperial aesthetic, dramatic ink wash cinematography,
silk robes and jade armor, golden light atmospheric"
```

## 九、制作流程（SLG 买量版本）

```
Step 1: 确定创意类型（6选1）
        ↓
Step 2: 确定题材（战争/末日/三国/魔幻）
        ↓
Step 3: 确定时长（15s 或 30s）和画幅（16:9 或 9:16）
        ↓
Step 4: 选择 HOOK 钩子类型
        ↓
Step 5: 套用对应脚本模板，填入游戏特定内容
        ↓
Step 6: 生成 Nana Banana Pro 资产清单（角色C/场景S/道具P）
        ↓
Step 7: 生成 Seedance 2.0 分镜提示词
        ↓
Step 8: 生图 → 生视频 → 剪辑合成 CTA
```
