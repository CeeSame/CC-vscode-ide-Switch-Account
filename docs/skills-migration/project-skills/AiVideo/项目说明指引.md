# AI 视频制作项目说明指引

> 基于 Claude Code + Nana Banana Pro + Seedance 2.0 的完整 AI 视频制作工作流

---

## 一、项目背景

本项目参考 Bilibili 视频《我用 AI 搭了一个分镜师团队，从剧本直出 Seedance 2.0 视频》（UP主：废才俱乐部Club，BV153PFzSEku），基于开源项目 [liangdabiao/Seedance2-Storyboard-Generator](https://github.com/liangdabiao/Seedance2-Storyboard-Generator) 搭建。

**核心理念：** 用 AI 组建一支"分镜师团队"，自动完成从故事 → 剧本 → 素材生成 → 分镜脚本 → 视频的全流程。

---

## 二、工具链

| 工具 | 用途 | 说明 |
|------|------|------|
| **Claude Code** | 剧本 + 分镜脚本生成 | 本项目核心，通过 Skill 驱动 |
| **Nana Banana Pro** | 角色/场景/道具图片生成 | 也可用其他生图 AI 替代 |
| **Seedance 2.0（即梦）** | AI 视频生成 | ByteDance 旗下平台 |

## 四、快速开始

### 第一步：启动 Claude Code

在 `AiVideo/` 目录下打开 Claude Code（终端或 VS Code 扩展）。

### 第二步：调用技能

```
/skill seedance-storyboard-generator
```

然后输入你的故事内容（可以是完整小说段落，也可以是简单大纲），例如：

```
请帮我把《司马光砸缸》改编成 5 集 AI 视频系列，水墨动画风格，9:16 竖屏
```

### 第三步：Claude 自动生成三份文件

| 文件 | 内容 |
|------|------|
| `[故事名]_剧本.md` | 完整四幕结构剧本（含人物小传、大纲、正文） |
| `[故事名]_素材清单.md` | C/S/P 编号的图片生成提示词 |
| `[故事名]_E01_分镜.md` | 第1集 Seedance 2.0 分镜脚本（含时间轴） |

## 五、完整制作流程

> 基于原始参考视频（BV153PFzSEku）工作流，增加了 AI导演审核 + 服化道确认 + 存档机制

```
阶段0  确认生成平台  →  可灵 / Seedance 2.0 → 调取对应提示词规范
           ↓
Step 1  输入故事     →  /skill seedance-storyboard-generator
           ↓
Step 2  确认制作参数  →  风格/集数/画幅/基调（参数确认阶段）
           ↓
Step 3  生成剧本     →  [故事名]_剧本.md（起承转合+人物小传+正文）
           ↓
Step 3.5  AI 导演审核  →  叙事节奏/人物一致/视觉可行/集数衔接
           ↓（用户确认通过）
Step 3.6  服化道确认  →  [故事名]_服化道设计表.md（角色/场景/道具精确定义）
           ↓（用户确认通过）
Step 4  生成素材清单  →  [故事名]_素材清单.md（C/S/P 编号提示词）
           ↓
Step 5  Nana Banana Pro 生图
         将素材清单中提示词逐条输入 → 保存为 C01.png / S01.png 等
           ↓
Step 6  生成分镜脚本  →  [故事名]_E01_分镜.md（平台对应格式提示词）
           ↓
Step 7  视频平台生成
         可灵：5s/10s clip + 截帧续写
         Seedance 2.0：15s + 视频延长
           ↓
Step 8  存档  → 说"存档"，生成 [故事名]_项目存档.md
```

## 六、素材编号规范

| 前缀 | 类型 | 示例 |
|------|------|------|
| C01–C99 | 角色（多角度） | C01 主角·正面全身，C02 主角·侧面 |
| S01–S99 | 场景/地点 | S01 草料场·雪景，S02 山神庙·夜景 |
| P01–P99 | 道具/物品 | P01 长枪，P02 酒葫芦 |

## 七、Seedance 2.0 重要限制

| 限制项 | 说明 |
|--------|------|
| 每次最多 9 张图片 | 超过则需分批上传 |
| 每次最多 3 个视频 | 总时长 ≤ 15 秒 |
| 提示词建议 ≤ 300 字 | 过长会导致指令遵循不稳定 |
| 避免敏感词 | 如"江湖人士"等，建议换一种说法 |
| 视频修改成本高 | 大多数改动需要完整重新生成 |

## 八、分镜格式速查

### 剧本正文格式（标准）

```
第X集
X-X [日/夜] [内/外] [场景名]
道具：XXX
出场人物：XXX

△ 【空镜】开场氛围镜头描述
△ 运镜描述 + 画面内容
角色名（os）：内心独白
角色名（情绪）：对白台词
△ 【闪回】回忆内容
【闪回结束】
【字幕：XXX】
```

### Seedance Prompt 格式（时间轴）

```
风格描述，时长，画幅，氛围

0-3s: 场景建立 - 镜头运动，画面描述
3-6s: 主体引入 - 镜头运动，动作描述
6-9s: 发展冲突 - 镜头运动，情绪表达
9-12s: 高潮转折 - 镜头运动，情感爆发
12-15s: 结尾余韵 - 镜头运动，最终画面

【声音】配乐风格 + 音效 + 对白
【参考】@图片1 用途，@图片2 用途
```
